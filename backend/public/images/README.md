# Carpeta de Imágenes - Landing Page

## Imagen de Fondo Principal

Para personalizar la landing page institucional (`index.html`), coloca tu imagen de publicidad aquí:

**Nombre del archivo**: `landing-bg.jpg`
**Ruta completa**: `public/images/landing-bg.jpg`

### Especificaciones Recomendadas

- **Formato**: JPG, JPEG o PNG
- **Resolución mínima**: 1920x1080 px (Full HD)
- **Resolución óptima**: 2560x1440 px o superior
- **Peso**: Máximo 500 KB (optimizada para web)
- **Orientación**: Horizontal (landscape)
- **Aspect Ratio**: 16:9 o 16:10

### Instrucciones

1. **Prepara tu imagen**:
   - Asegúrate que tenga buena resolución
   - Optimízala para web (comprime si es muy pesada)
   - Verifica que el contenido importante esté centrado

2. **Renombra el archivo**:
   - El nombre DEBE ser exactamente: `landing-bg.jpg`
   - Minúsculas y sin espacios

3. **Coloca el archivo**:
   - Copia/pega tu imagen en esta carpeta
   - Ruta: `C:\Bio\sistema_asistencia_biometrico\backend\public\images\`

4. **Verifica**:
   - Abre tu navegador
   - Ve a: `http://localhost:9998/`
   - Deberías ver tu imagen de fondo

### Fallback

Si NO colocas ninguna imagen, la landing page usará automáticamente un gradiente morado como fondo (no rompe nada, solo se ve diferente).

### Otras Imágenes

Puedes agregar más imágenes en esta carpeta y referenciarlas desde el HTML como:

```html
<img src="/images/nombre-imagen.jpg" alt="Descripción">
```

---

**Tip**: Para mejores resultados, usa imágenes con áreas oscuras o semitransparentes donde irá el texto, para asegurar buena legibilidad.
